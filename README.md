# password_keeper
  Based on Bootstrap and jQuery,and thank them to simplify my codes.
## quick start!
  Ensure you have python and pip
  
  Use
  ~~~bash
  git clone https://github.com/Codecsystem/password_keeper.git
  ~~~
  then Use
  ~~~bash
  pip install flask
  pip install flask_cors
  pip install Crypto
  ~~~
  then Use 
  ~~~bash
  cd "your catelogue"
  python -u server.py
  ~~~
